# BIAN Resource Management Service

Ballerina package implementing the BIAN **Resource Management** service domain.

## Package

| Property | Value |
|----------|-------|
| Org | `ballerinax` |
| Name | `resource.management` |
| Version | `0.1.0` |
| Default port | `9097` |

## Services

No services are currently registered. This package is a placeholder for future Resource Management service domain implementations.

## Configuration

Edit `Config.toml` to point at your backend:

```toml
port = 9097
serverUrl = "http://localhost:9092/backend"
```

## Running
